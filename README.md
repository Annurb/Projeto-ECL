# Otimização de Circuitos Integrados utilizando Redes Neurais Artificiais

Este repositório contém o desenvolvimento e os experimentos realizados durante o meu período como Bolsista Voluntária de Pesquisa na **Universidade Federal de Santa Catarina** no Embbed Computing Laboratory. 

O objetivo principal do projeto foi aplicar técnicas de Inteligência Artificial para otimizar o design e o desempenho de circuitos integrados.

## 🛠️ Contribuições e Escopo Técnico
No ecossistema deste projeto, minhas principais contribuições focaram em:
* **Modelagem Preditiva:** Desenvolvimento de uma rede neural *Multilayer Perceptron* (MLP) *multi-output* com o objetivo de prever dois *targets* de engenharia simultaneamente, utilizando `scikit-learn`.
* **Tuning de Hiperparâmetros:** Implementação e execução de rotinas de *Grid Search* utilizando a biblioteca `scikit-learn` para identificar a combinação ideal de parâmetros da rede.
* **Exploração de Arquiteturas:** Análise comparativa de diferentes topologias de rede para ganho de eficiência computacional e precisão do modelo.
